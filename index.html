⚙ One-time setup

  1. Go to console.cloud.google.com → create a project
  2. Enable Google Drive API under APIs & Services
  3. Create an OAuth 2.0 Client ID (Web application type)
  4. Add http://localhost or your hosted URL to Authorised JavaScript origins
  5. Add yourself as a Test user on the OAuth consent screen
  6. Paste your Client ID below and click Save
